The theme admin section is divided into 4 fully customizable parts: Theme options, Default layout, Page Builder, Import / Export. Elements are available below Theme Admin options. You can manage from there all theme custom post types and features provided by elements which are ready to use in the theme. Working with theme is easy as following steps:

  • Create as many as you need pages in section Pages
  • Go to Page builder and select Page you wish to customize
  • Write elements content
  • Define page layout by drag and drop operations
  • Set up parameters of elements
  • Repeat previous steps
